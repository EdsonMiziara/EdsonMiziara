### 🚀 Principais Projetos em Destaque

#### 💳 [Malote Digital](https://github.com/EdsonMiziara/MaloteDigital) *(Projeto Independente / Solução Comercial)*
Sistema de alto desempenho voltado para a gestão financeira e automação de fluxos de caixa condominiais.
* **Destaques Técnicos:** Desenvolvido sob os princípios da **Clean Architecture** e *Rich Domain Model*. Implementa um motor autônomo de conciliação bancária via processamento de arquivos **OFX** com margem de tolerância e garante a segurança contra duplicidade gerando hashes criptográficos (**SHA-256**). 
* **Automação:** Possui um `BackgroundService` assíncrono para clonagem de despesas recorrentes a cada virada de mês e pipeline automatizado de **CI/CD via GitHub Actions** para testes automáticos (xUnit).
* *Tech Stack:* C#, .NET 8, Minimal APIs, EF Core, PostgreSQL, GitHub Actions, xUnit e FluentAssertions.

#### 🎵 [ScreenSound](https://github.com/EdsonMiziara/ScreenSound)
Plataforma completa para gerenciamento de artistas, músicas e álbuns.
* **Destaques Técnicos:** Separação estrita de camadas (Domínio, Dados, Requests/Responses), persistência com SQL Server via EF Core com migrations e desenvolvimento de interface interativa em **Blazor** integrada com controle de autenticação e regras de acesso.
* *Tech Stack:* C#, ASP.NET Core, Minimal APIs, EF Core, SQL Server, Blazor, LINQ, xUnit e Azure.

#### 📊 [Financial Control](https://github.com/EdsonMiziara/FinancialControl)
Ecossistema modular focado no processamento de transações e automação de tarefas financeiras.
* **Destaques Técnicos:** Arquitetura distribuída integrando API, Console App, Desktop App e **Azure Functions**. Conta com rotinas automáticas de categorização de transações, geração/automação de relatórios em Excel e integração com o Google Drive, tudo orquestrado e distribuído via **Docker**.
* *Tech Stack:* C#, .NET, ASP.NET Core, EF Core, SQL Server, Azure Functions, Docker e Excel Automation.
